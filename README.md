# Pang

A subset of scheme, implemented in F#.
Uses fslex and fsyacc.

## Supports

* Quasiquotation
* Recursive functions
* Anonymous functions

## Implemented

`if`, `let`, `let*`, `lambda`, `quote`, `define`, `set!`

## Functions implemented

### Input/Output

`display`

### Arithmetics

`+`, `-`, `*`, `/`

### Cons/List operations

`car`, `cdr`, `list`, `length`, `cons`

### Comparision

`eq`

### Functional programming

`apply`, `map`
